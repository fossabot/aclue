Welcome to a.clue

Fancy icons and symbols for your great web project. Easy to use and uses the popular web font FontAwesome.

What this is

Use the sexy glyphs from the popular web font FontAwesome whitout a lot of work. only you declare it once and the icons come on a magic way in your HTML document

How it work

a.clue use only CSS for the most icons and dont need necesary JavaScript, only to find out the host directory. Whatever it works fine whitout JS

How to use

Include the CSS in the section in your HTML document.

Add the aclue class at the parent block element you want.

You can overwrite the colors and style of all the icons in your own CSS file.

Lists with icons you make it so easy

Work also fine with Twitter Bootstrap.
